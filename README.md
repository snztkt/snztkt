### 自己紹介

2000年からインターネットの現場に身を置き、インフラの構築とソフトウェア開発を続けてきました。長野県の中信地方を拠点としています。  
現在は開発の最前線からは一歩退き、長年の経験に基づく「安定した設計」と、生成AIという「最新の道具」を組み合わせることで、身近な課題を形にすることを楽しんでいます。

### 技術領域と経験

計算機とネットワークの低レイヤーに深く関わってきました。

* **基盤・運用（25年以上）**
    * 主戦場は **FreeBSD** です。2000年から2011年までは、ISPにてサーバ運用とスタッフへの技術支援、利用者向けのヘルプデスクを担当していました。
    * ネットワークインフラの設計から、各種サーバの構築、日々の保守運用まで一通りを担います。

* **ソフトウェア開発（15年以上）**
    * **C言語** による開発を主軸としてきました。ハードウェアに近い領域での堅牢な実装を信条としています。
    * シェルスクリプトを用いた運用自動化や、SQLによるデータベース操作も実務に組み込んでいます。

* **近年の取り組み（還暦からのウェブ開発）**
    * 生成AIを技術パートナーとして活用したウェブ開発に着手。**JavaScript** / **Vue.js** を用いた実装を行っています。
    * 自身のエンジニアリング経験を土台に、AIを補完的に使いこなすことで、迅速にプロダクトを形にすることを目指しています。

### 制作物

#### NamPod（ナンポッド）
日蓮仏法の勤行（妙法蓮華経の読誦）を、子どもたちが親しみながら習得することを目的としたウェブアプリです。  
映画『F1（エフワン）』に登場したトレーニングツール「BlazePod」の視覚的なリズム感にインスパイアされ、伝統的な修行を現代的なゲーム形式へと再構築しました。

* **公開URL:** https://snztkt.github.io/nam-pod/
* **開発のこだわり:**
    * 経文に使用される旧字体を忠実に再現するため、Google/AdobeのCJKフォントを採用。
    * 約900文字から成る経文の全文字を音声データ化（Google TTS）し、一音ずつのガイドを実装。
    * **色覚多様性への配慮:** 赤系・緑系の色を避け、誰もが判別しやすい配色と簡潔なデザイン。
    * 生成AIとの対話を通じて、長年培ったエンジニアの勘をモダンなUIへと落とし込みました。
